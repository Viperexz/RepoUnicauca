## Requisitos:
- Base de conocimiento: Se debe integrar los archivos en la carpeta data para su procesamiento por los embedings y almacenados los chunks en la base de datos vectorial.
- Conexion a internet: Se generar los embeding con el modelo <code> paraphrase-multilingual-MiniLM-L12-v2 </code> de HugginFace.
- Ollama: El RAG trabaja con el modelo <code> Mistral:7B </code> por lo que se necesita tener a ollama funcionando y con el modelo instalado.


## Ejecucion:

El archivo main.py ejecuta el proceso principal.
